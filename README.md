
## 실험용 레포 
React Query의 paginated query, infinite query실험용 간단쇼핑몰입니다.


## 사용순서

첫째, 'yarn server'를 통해 json-server 활성화


둘째,
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```


셋째, [http://localhost:3000](http://localhost:3000) 열기
